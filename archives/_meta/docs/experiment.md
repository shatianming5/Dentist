# Experiments

## Summary

- Main goal: journal-ready, reproducible baselines + reporting artifacts for dental point-cloud tasks in this repo.
- Baseline: raw_cls (PointNet/DGCNN), Teeth3DS synthetic prep→target + constraints (margin/occlusion proxies).
- Main model: (this repo scope) baselines + protocol tooling; proposed models live in `plan_report.md` but are **not claimed** (see `PAPER_SCOPE.md`).
