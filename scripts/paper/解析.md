# scripts/paper 解析

这里放 **论文表格/报告生成** 相关模块与脚本。

目标：

- 把“从 runs/paper_tables 读数据 → 聚合 → 输出 markdown/csv”这类逻辑拆成可复用模块
- 入口脚本尽量只做参数解析与调度

## 典型入口（原 scripts 根目录仍保留兼容）

- `scripts/paper_table_raw_cls.py`：raw_cls 表格
- `scripts/paper_raw_cls_kfold_merged_report.py`：kfold 汇总报告
- `scripts/paper_raw_cls_multi_ensemble.py`：mean-prob 集成
- `scripts/paper_raw_cls_stacking_ensemble.py`：stacking/weighted 集成
- `scripts/paper_raw_cls_domain_shift_table.py`：domain shift 表格
