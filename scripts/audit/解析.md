# scripts/audit 解析

这里放 **审计/自检类脚本与模块**（尽量不涉及训练/产生 runs），用于：

- 检查数据一致性/协议合规
- 检查论文表格与 runs 的对应关系
- 定位常见误差模式（比如 raw_cls 的混淆）

## 典型入口（原 scripts 根目录仍保留兼容）

- `scripts/docs_audit.py`：docs-spec 闭环自检（已归档 docs 在 `archives/_meta/docs/`）
- `scripts/paper_audit.py`：论文范围/协议审计
- `scripts/journal_audit.py`：期刊自检报告
- `scripts/raw_cls_ensemble_error_audit.py`：raw_cls 集成误差审计

## 约束提醒

- 本目录应避免写入 `runs/`、`paper_tables/` 等实验产物；需要输出时优先打印到 stdout 或写到 `archives/_meta/` 下的 debug 位置。
