# scripts/data_tools 解析

这里放 **数据处理/统计/校验** 相关工具，特点是：

- 读取 `raw/`, `converted/`, `processed/`, `metadata/` 等
- 尽量不写入 `runs/`（除非明确是训练脚本）

## 典型入口（原 scripts 根目录仍保留兼容）

- `scripts/convert_ccb2_bin.py`：raw 转 bin/npz
- `scripts/label_converted_raw.py`：给 converted/raw 贴标签
- `scripts/validate_converted_raw.py`：检查 converted 数据合法性
- `scripts/dataset_stats.py`、`scripts/raw_dataset_stats.py`：数据统计
- `scripts/make_raw_kfold_splits.py`：切 kfold
