# scripts/teeth3ds 解析

这里放 **teeth3ds** 相关的模块与入口拆解，通常包含：

- 数据构建/修复（split、对齐检查等）
- 预训练（AE）
- 牙位分类（FDI 分类）
- 与 prep2target 的对接

## 典型入口（原 scripts 根目录仍保留兼容）

- `scripts/phase2_build_teeth3ds_teeth.py`
- `scripts/phase2_train_teeth3ds_ae.py`
- `scripts/phase2_train_teeth3ds_fdi_cls.py`
- `scripts/fix_teeth3ds_teeth_splits.py`
- `scripts/check_teeth3ds_jaw_alignment.py`

## 模块索引（用于拆解复用）

- `scripts/teeth3ds/obj_io.py`：OBJ 顶点解析（只读 `v x y z`，可提前在 faces 处停止）
- `scripts/teeth3ds/constraints_data.py`：constraints 相关数据结构（dataset/collate + opposing cache + jaw/fdi 工具）
- `scripts/teeth3ds/prep2target_dataset.py`：phase3 的 teeth3ds `prep2target` 数据集封装
